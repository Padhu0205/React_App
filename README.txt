# MERN Todo App

A complete Todo List application built using MongoDB, Express.js, React.js, and Node.js (MERN stack).
Includes user authentication (JWT), CRUD operations, and Context API for state management.

## Steps to Run

1. Install dependencies:

```bash
cd backend
npm install

cd ../frontend
npm install
```

2. Create `.env` file in backend folder:
```
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret
PORT=5000
```

3. Run backend and frontend:
```bash
npm run dev   # backend
npm start     # frontend
```
